DXF Import .NET is c# code to read AutoCAD DXF files. It supports:
LINE
TEXT
CIRCLE
ELLIPSE
BLOCK
INSERT

entities.
DXF Import .NET is opensource project which shows a way to read AutoCAD DXF files.
